# InsomniMac 1.0

The first public release of InsomniMac, a small menu bar utility that keeps a Mac reachable while remote-control agents are working.

## Highlights

- Toggle Awake Lock from the menu bar or with `Command-Shift-Backslash`.
- Keep the Mac available for remote-control agents while you are away.
- Optionally show the animated awake overlay while the lock is active.
- Launch automatically at login and optionally prevent sleep with the lid closed.
- Share Remote Mode context with supported agents.
- Automatically check for, securely verify, and install future updates.

InsomniMac is free, open source, and MIT licensed.
